2018年终总结

Posted on 2018-12-30

​
转眼间，2018年已经过去，是时候总结一年的得失了。

关于工作

  • 年初的时候一个人跑来了上海，找工作自然地在BOSS直聘以及前程无忧海投简历。很庆幸自己在能够面试几家后成功入职。
  • 新公司业务偏物联网领域，有些行业名词初期不熟悉做功能的时候比较容易碰壁，有段时间压力还是比较大。
  • 国庆节后被拉去杭州出差，感受了下杭州那边的加班文化。
  • 跟随大佬重构了某海外项目中的部分核心代码，过程中学习到了很多良好的编码习惯以及软件构架思想。

关于技术

  • 在这一年中掌握了 Angular 全家桶的基本用法，并且可以在项目中正常地进行使用。
  • 增强了自己的 Linux 操作能力，特别是下半年自己部署的一个模块老是出现各种 bug ，在进行线上排查的时候对平时不是很熟悉的命令能够更加熟练。
  • 技术栈依旧是以 Java 为主要编程语言以及常用的消息中间件，流行框架等。
  • 学习了 Docker 的使用，并且能进行简单的运用。
  • Github 今年好像一年都没更新过，好懒。。。。。
  • 更加熟练地运用 Java 并发相关 API 。
  • 业余时间尝试使用 Python 进行爬虫的编写。
  • 继续学习了安卓逆向的知识，也自己上手破解了几个小软件，也有顺便去横幅广告之类的，成就感还是满满的。

关于生活

  • 共计跑步 23 次，每次时间平均为 1 小时 。平均算下来从 6 月份 到 10月份间，每周大概 2 次 。每次跑完步之后整个人都感觉到了一股神奇的力量。
  • 共计打王者农药 800 场左右 ，一般就是周末代码写的久了然后也确实挺无聊的，一不小心一年下来就打了这么多场，还好现在已经脱坑了。
  • 共计钓鱼 11 次，钓上来的鱼倒是少的可怜，但是别的东西倒是一大堆。
  • 学习完了 新概念英语3 后面的 21 篇文章。
  • 在百词斩中累计打卡 180 天，累计背单词 6734 个，刷完了 考研英语 以及 六级词汇 两本单词书。
  • 累计在扇贝阅读中打卡 249 天。

总结

在技术上，这一年比较盲目地扩张技术的广度，忽视了技术的深度学习。这是一个技术从业者比较致命的地方，很多比较流行的新技术，例如 Spark，Hadoop，Strom等等，这些在目前工作中还暂时使用不到。

在沟通能力上，满分有10分的话，我给自己评分可以给个7分左右，在工作中跟同事以及领导沟通是没有问题的。需要改善的是能够抓住重点，一针见血地指出问题。

在生活上，体育锻炼相比 2017年确实是少了不少。这个还是需要加强的，程序员还是需要多加锻炼身体的。

在技能学习上，今年给自己打个差评。在英语的学习上有了松懈，对语法没有及时的巩固，并且没有去运用。导致的结果就是写起句子以及文章来还是一塌糊涂。在口语练习上没有明显进步，依旧是哑巴水平。

今年总的来说一年过得非常的空虚，最主要的可能是自己内心没有一个明确的目标，没能朝着这个目标前进。晚上躺着床上的时候总会在想，自己究竟要什么，我需要做什么才能得到我想要的。

2019年目标

  • 加强技术的深度学习，多往部分技术底层学习，不盲目追求技术的广度。
  • 加强英语写作能力以及口语能力的练习。
  • 加强沟通交流能力。
  • 多加锻炼身体，身体才是革命的本钱。
  • 多把握机会，这个就不方便透露了。。。。
